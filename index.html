CC3200 Lab Project Demo

Features

Login

Use the IR remote for text/numeric input on the login page. Enter your username and password using T9-style text messaging.

Main Menu

After successful login, navigate to the main menu page. Press different buttons to select and play different games.

Games Implemented

Tic Tac Toe

Play Tic Tac Toe with two players. Use the remote control to select squares on the grid and place Xs or Os. The game determines the winner or a draw, and returns to the main menu page.

Number Memory

Memorize and enter displayed numbers. A correct guess increases the score and adds another number to memorize. An incorrect guess ends the game and sends the score through AWS.

Word Memory

Remember whether you have seen a word before. Keep score and lives. Use the SW3 and SW2 switches to indicate new and seen words. Once lives run out, the score is sent through AWS.

Battleship

Play Battleship on a 4x4 grid with UART communication between LaunchPads. Place ships and attack the opponent's grid. Hits and misses are visually indicated. When all ships are sunk, messages are sent via UART and AWS.
